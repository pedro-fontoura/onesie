Solvna

From the game designer:

Soluna is a single player divination game, using cards 1 to 6 from a traditional deck. You must successfully complete eight predictions before the deck runs out or all options are blocked.

It was designed to be distributed as an A4 zine, with rules and game board on either side. So far I only made a french version of this zine, that can be checked here. The whole zine has been put together without the use of a computer. All the texts were written with a typewriter, and the illustrations were sourced from an astrology books. Everything was put together with a copier.

This game is a submission for the BGG's 2025 Traditional Deck Game Design Contest.

Rules in english are available here.

– Jean-Baptiste Lévêque
